# HR Attrition Analysis Dashboard (Power BI)

## Overview
This project explores employee attrition patterns using an interactive Power BI dashboard, focusing on identifying high-risk roles and workforce segments to support data-driven retention decisions.

## Tools Used
- Power BI
- DAX

## Key Insights
- Attrition is concentrated in specific roles, particularly Laboratory Technicians, Sales Executives, and Research Scientists.
- Employees working overtime show consistently higher attrition, especially within lower-to-mid income bands.
- Combining job role, overtime status, and income highlights clear high-risk employee segments suitable for targeted retention strategies.

## Deliverables
- Interactive Power BI dashboard
- Role- and segment-level attrition analysis
